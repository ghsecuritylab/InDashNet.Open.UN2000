Release Note:

[更新列表]
1、Android版本升级到Android4.4。
2、兼容GMS，内部测试0 fail。
3、升级到boot2.0，优化启动流程。
4、升级到nand2.0，增强稳定性，提升随机读写速度。
5、新增miracast内存自适应功能，提升流畅度。
6、支持平板加密功能。
7、更新WiFi和GPU驱动。
